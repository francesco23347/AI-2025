## 🚀 ฟีเจอร์หลัก (Features)
• AI Matching System – ใช้ Machine Learning เพื่อเลือกอาสาสมัครที่เหมาะสมที่สุด

• Service Filtering – ค้นหาอาสาสมัครตามประเภทบริการ

• Distance Filtering – จำกัดระยะทางของอาสาสมัคร

• Smart Ranking – จัดอันดับอาสาสมัครด้วย AI Score

• Data Dashboard – แสดงข้อมูลผ่านกราฟและตาราง

1. สิ่งที่ได้เรียนรู้
จากการทำโครงงานครั้งนี้ ได้เรียนรู้เกี่ยวกับการพัฒนาโปรแกรมโดยนำปัญญาประดิษฐ์ (AI) มาประยุกต์ใช้ในการแก้ปัญหา โดยเฉพาะการใช้โมเดลในการช่วยวิเคราะห์และจัดอันดับข้อมูล นอกจากนี้ยังได้เรียนรู้การเขียนโปรแกรมด้วยภาษา Python การใช้ไลบรารีต่าง ๆ เช่น Pandas, Streamlit และการจัดการข้อมูลจากไฟล์ Excel/CSV รวมถึงการออกแบบระบบให้สามารถนำข้อมูลมาใช้ในการตัดสินใจและแสดงผลออกมาในรูปแบบที่เข้าใจง่าย

2. สิ่งที่ทำได้ดี
ในการทำโครงงานครั้งนี้ สามารถพัฒนาระบบให้สามารถทำงานได้ตามแนวคิดที่วางไว้ เช่น การสร้างระบบจับคู่อาสาสมัครกับประเภทการช่วยเหลือ การนำ AI มาช่วยคำนวณคะแนนเพื่อจัดอันดับอาสาสมัคร และการออกแบบหน้าเว็บให้สามารถใช้งานได้ง่าย นอกจากนี้ยังสามารถแก้ไขปัญหาที่เกิดขึ้นระหว่างการเขียนโปรแกรม และปรับปรุงระบบให้ทำงานได้ถูกต้องมากขึ้น

3. สิ่งที่ต้องปรับปรุงพัฒนาในอนาคต
ในอนาคตควรพัฒนาความรู้เกี่ยวกับ AI และโมเดล Machine Learning ให้มากยิ่งขึ้น เพื่อให้สามารถเลือกใช้โมเดลที่เหมาะสมและมีประสิทธิภาพมากขึ้น รวมถึงควรปรับปรุงการออกแบบระบบให้มีความแม่นยำและรองรับข้อมูลได้มากขึ้น นอกจากนี้ควรเพิ่มความรอบคอบในการเขียนโปรแกรมและการจัดการข้อมูล รวมทั้งศึกษาค้นคว้าข้อมูลจากแหล่งต่าง ๆ ให้กว้างและลึกมากขึ้น เพื่อนำความรู้ไปพัฒนาระบบให้ดียิ่งขึ้นในอนาคต.
